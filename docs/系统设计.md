## 技术选型

| 组件 | 选择 | 理由 |
|------|------|------|
| 后端框架 | FastAPI | 异步支持，自动API文档 |
| 前端框架 | React + Antd | 组件丰富，中文生态好 |
| 图谱可视化 | ECharts | 赛题推荐，支持力导向图 |
| 向量数据库 | ChromaDB | 轻量，易集成 |
| 图数据库 | NetworkX | Python原生，适合小规模图 |
| Embedding | paraphrase-multilingual-MiniLM-L12-v2 | 支持中文，赛题建议 |

## API 接口一览

| 方法 | 路径 | 功能 |
|------|------|------|
| POST | /api/upload | 文件上传 |
| GET | /api/books | 教材列表 |
| GET | /api/books/{id} | 教材详情 |
| POST | /api/books/{id}/build-kg | 构建图谱 |
| GET | /api/books/{id}/kg | 获取图谱 |
| GET | /api/kg/merged | 合并图谱 |
| POST | /api/merge | 触发整合 |
| GET | /api/merge/decisions | 整合决策 |
| POST | /api/compress | 触发压缩 |
| GET | /api/compress/stats | 压缩统计 |
| POST | /api/rag/query | RAG问答 |
| POST | /api/teacher/chat | 教师对话 |
