# 接口文档

## 文件上传

### POST /api/upload

上传教材文件，支持 PDF、DOCX、MD、TXT 格式。

**请求**：multipart/form-data
- files: 文件列表

**响应**：
```json
{
  "textbooks": [
    {
      "textbook_id": "book_0001",
      "filename": "病理学.pdf",
      "title": "病理学",
      "total_chars": 50000,
      "chapters": 10
    }
  ]
}
```

## 教材管理

### GET /api/books

获取教材列表。

**响应**：
```json
{
  "books": [...]
}
```

### GET /api/books/{book_id}

获取教材详情。

### POST /api/books/{book_id}/build-kg

构建知识图谱。

**响应**：
```json
{
  "status": "completed",
  "nodes": 50
}
```

### GET /api/books/{book_id}/kg

获取知识图谱 ECharts 数据。

## 知识图谱

### GET /api/kg/merged

获取合并后的知识图谱。

## 跨教材整合

### POST /api/merge

触发跨教材整合。

### GET /api/merge/decisions

获取整合决策列表。

**响应**：
```json
{
  "decisions": [
    {
      "decision_id": "merge_001",
      "action": "merge",
      "affected_nodes": ["node_001", "node_002"],
      "result_node": "node_001",
      "reason": "语义等价",
      "confidence": 0.85
    }
  ]
}
```

## 内容压缩

### POST /api/compress

触发内容压缩。

### GET /api/compress/stats

获取压缩统计。

**响应**：
```json
{
  "original_chars": 100000,
  "compressed_chars": 25000,
  "ratio": 0.25
}
```

## RAG 问答

### POST /api/rag/query

RAG 问答。

**请求**：
```json
{
  "question": "什么是炎症？"
}
```

**响应**：
```json
{
  "answer": "炎症是机体对致炎因子的防御反应...",
  "citations": [
    {
      "textbook": "病理学",
      "chapter": "第四章 炎症",
      "page": 78,
      "relevance_score": 0.92
    }
  ],
  "source_chunks": [...]
}
```

## 教师对话

### POST /api/teacher/chat

教师对话。

**请求**：
```json
{
  "message": "为什么合并了X和Y？",
  "session_id": "session_001"
}
```

**响应**：
```json
{
  "reply": "因为两者描述的是同一概念...",
  "updated_decisions": [...]
}
```
