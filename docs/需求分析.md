# 需求分析

## 知识点粒度定义

知识点定义为教材中的核心概念、定理、方法或现象。粒度标准：
- **核心概念**：独立的学科术语或概念
- **定理**：可证明的命题或公式
- **方法**：解决问题的步骤或策略
- **现象**：可观察的学科现象

## 重复判定标准

采用双重对齐策略：
1. **Embedding粗筛**：计算 name+definition 的余弦相似度 > 0.85
2. **LLM精判**：对候选对调用 LLM 判断语义等价性

## 教学连贯性保障

- 保留 prerequisite 关系链
- 合并时保留描述最系统的版本
- 标注所有来源教材

## 压缩比计算

```
压缩比 = 整合后字数 / 原始总字数
目标：≤ 30%
```

## RAG 分块策略

- 分块大小：600字 (赛题要求500-800字)
- 重叠大小：80字 (赛题要求50-100字)
- 选择中间值平衡精度和效率
